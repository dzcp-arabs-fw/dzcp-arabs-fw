Creating a Basic Plugin

To create a plugin in DZCP, follow this example:

using DZCP.Bootstrap;
using DZCP.Core;

public class MyPlugin : IPlugin
{
    public void OnLoad()
    {
        Console.WriteLine("[MyPlugin] Plugin loaded successfully!");
    }
}

List of Plugins

  • 🎉 Welcome Plugin - Sends a welcome message to new players.
  • ⚰️ Death Logger - Logs player deaths in the console.
  • 🔮 SCP Ability Logger - Tracks SCP ability usage.
  • 📢 Server Announcements - Allows admins to send messages.
