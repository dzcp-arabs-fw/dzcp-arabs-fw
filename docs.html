⚙️ How to Create a Plugin


using DZCP.Framework;
using DZCP.Framework.API;
using DZCP.Framework.Events;
using DZCP.Framework.Plugins;

public class MyPlugin : IPlugin
{
    public string Name => "My Plugin";
    public string Version => "1.0.0";

    public void OnLoad()
    {
        EventManager.RegisterListener("PlayerJoin", new PlayerJoinHandler());
    }
}

The code above is a simple example of how to register an event when a player joins the server.

📌 Core Concepts

  • IPlugin: The base interface for creating plugins.
  • EventManager: Manages in-game events dynamically.
  • API: Provides functions for interacting with the game.
  • PlayerJoinHandler: An example of handling player join events.
